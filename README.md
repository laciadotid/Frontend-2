## Rekomendasi sistem

Node.js 18 / LTS terbaru

## jika belum mengaktifkan Yarn

```bash
corepack enable
```

## instalasi

```bash
git clone https://gitlab.com/laciadotid/Frontend.git
```

```bash
cd Frontend
```

```bash
cp .env.example .env
```

```bash
yarn
```

```bash
yarn prepare
```

## deploy

```bash
yarn build
```

publish directory : out
